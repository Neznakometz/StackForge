# saas — доменный набор

Бэкенд/веб-сервисы поверх ядра Claud Framework.

**Скилы:** `api-design` · `background-jobs` · `multitenancy`.
**Из ядра:** `contracts`, `task-loop`, `cross-model-review`, `security-auditor`.
**Стек-знания (через `/init`):** knowledge-паки `php-laravel` / `python-django` / `node-nestjs`, `data-sql`, `data-cache`, `data-search-olap`.

**Внешние скилы (опц., ставятся отдельно):**
- `postgres-best-practices` (Supabase) — индексы, RLS, zero-downtime миграции.
- `mcp-builder` (Anthropic) — построить MCP-сервер к внутреннему API.
