- итог: PASS / FAIL + счётчики;
- при FAIL: до 5 первых уникальных ошибок — файл:строка + однострочное сообщение + наиболее вероятная причина;
- НЕ вставляй полные стектрейсы и логи. Ответ ≤20 строк.
