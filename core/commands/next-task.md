Выполни следующую невыполненную задачу текущей фазы по скилу `task-loop`:

1. Прочитай `PROGRESS.md`, определи текущую фазу и следующую задачу из плана реализации.
2. Если задача крупная — разбей на подзадачи 5–15 мин и зафиксируй разбиение в PROGRESS.md.
3. Прогони цикл: `implementer` → `spec-reviewer` → `code-reviewer` (+`design-reviewer` для UI) → `cross-model-review` → `test-runner`.
4. Обнови PROGRESS.md и покажи краткий итог (≤10 строк).

Если задача требует решения, не описанного в документах, — остановись и спроси, варианты не выдумывай.
