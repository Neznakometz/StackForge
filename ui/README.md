# ui — доменный набор

Дизайн интерфейсов поверх ядра Claud Framework.

**Скилы:** `ui-architecture` · `design-system` · `web-a11y`.
**Из ядра:** `design-reviewer`, `task-loop`, `cross-model-review`.
**Стек-знания (через `/init`):** `frontend-react` или `frontend-vue-nuxt`, `frontend-styling`.

**Внешние скилы (опц., ставятся отдельно):**
- `frontend-design` (Anthropic) — Claude как дизайнер на UI-задачах.
- `web-design-guidelines` + `react-best-practices` (Vercel) — аудит UI и React-анти-паттернов.
- `shadcn-ui` — project-aware генерация компонентов из `components.json`.
- `canvas-design` (ComposioHQ) — постеры/обложки/инфографика кодом.
